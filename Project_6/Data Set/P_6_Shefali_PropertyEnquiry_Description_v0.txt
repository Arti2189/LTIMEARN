                                                          REAL ESTATE ENQUIRY
                                                                
                                                                
This project is based on Real Estate platform. It aims at providing information regarding residential properties available for the purpose of buy or rent. A visitor can search for his ideal house/houses on multiple criterion such as county, city, no. of bedrooms/bathrooms etc. as per his requirements.



FIELDS INCLUDED

"Listing Office - Office Name" : represents the name of the office under which the property is listed
"List Agent - Agent Name" : represents the name of the Agent associated
"Price" : Price of the property
"Status" : indicates availability of the property, whether open or closed
"Address" : Address of the property
"City" : City in which the property is located
"MLS#" : Multi-Listing Service; proprietary information of the broker who has obtained a listing agreement with a property's seller.
"Property Type" : represents the parties for which the property is available(for eg:, Single Family)
"State" : State in which the property is located.
"County" : County in which the property is located
"Date - MLS" : Date on which the agreement is listed under MLS
"Bedrooms - Total" : No. of bedrooms in the corresponding property
"Baths - Total" : No. of bathrooms in the corresponding property
"SqFtTotFn" : Area of the property in square feet
"DOM" :  represents Days on Market, i.e. for how many days a property has been available for sale
"Assessment Amount" : represents the dollar value assigned to a property to measure applicable taxes
"Assessment Year" : represents the year that forms the basis for taxation in the next or following year
"Tax - Gross Amount" : represents the amount of tax applied on that property
"Lot - Acres" : Area of the property in acres
"Garage" : specifies if garage is available or not
"Year Built"  : represents the year in which the property is built
"Total Stories"  : represents the no. of stories of a property
"Surveyed" : specifies if the property has been surveyed earlier
"Seasonal"  : specifies if the property is seasonal, i.e. available for particular seasons only
"Rooms - Total"  : represents total no. of rooms in a property
"Garage Capacity"  : specifies the maximum no. of vehicles that can be kept in the garage
"Basement" : specifies whether a basement is available or not

Please follow this instructions:

1.Unzip JobsPortal.zip
2.Go to this folder using cmd
3. Run command -npm install(This will install all the depedencies)
4.Remain in same folder and run 'ng serve --proxy-config proxy.config.json'
5.Open browser and run 'http://localhost:4200'

/*If everything is working fine you will see index page of JobsPortal*/
